LIBRARY:
The first line is a library, which allows us to use different pre-defined functions in the code.

INT MAIN():
The second line opens up a section for us to write our code in it.

DECLARATION OF VARIABLES:
The third line declares three variables of integer type which will be used later on.

PRINT COMMAND:
The fourth line is used to print that line on the console window.

SCANF COMMAND:
The fifth line takes input from the user and stores it in our declared variables.

SUMMATION AND STORAGE:
In the sixth line, summation of inputted two numbers in stored in our third declared variable.

OUTPUT:
The seventh line prints the summation of the numbers on the console window.

RETURN 0:
And the last line is used to indicate the successful execution of the program.
